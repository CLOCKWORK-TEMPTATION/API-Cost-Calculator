# Gemini API Cost Calculator | حاسبة تكلفة Gemini API

تطبيق ويب مبني باستخدام React يساعد المطورين على حساب وتقدير تكاليف استخدام نماذج Google Gemini API المختلفة. يوفر التطبيق واجهة لحساب التكلفة يدوياً بالإضافة إلى تجربة حية لقياس استهلاك التوكن الفعلي.

## المميزات الرئيسية

1.  **حاسبة يدوية (Manual Calculator)**:
    *   تقدير التكلفة بناءً على عدد المدخلات (Input Tokens) والمخرجات (Output Tokens).
    *   **جديد:** أداة تحويل "عدد الكلمات" إلى توكنز للمساعدة في التقدير.
    *   مقارنة رسومية بين النماذج المختلفة.
    *   تنسيق ذكي للعملات (Micro-cents) لعرض التكاليف الدقيقة.

2.  **تجربة حية (Live Estimator)**:
    *   إرسال نصوص حقيقية إلى Gemini API.
    *   حساب عدد التوكنز الفعلي للمدخلات والمخرجات.
    *   عرض التكلفة الدقيقة للطلب بناءً على تسعير النموذج المختار.

3.  **إدارة النماذج (Model Management) مع الحفظ التلقائي**:
    *   إمكانية إضافة نماذج مخصصة (Custom Models).
    *   **بحث ذكي (Auto Search)**: يستخدم التطبيق Gemini للبحث في الويب عن أحدث أسعار ومواصفات النماذج وتعبئة البيانات تلقائياً.
    *   **الحفظ المحلي (Persistence)**: يتم حفظ النماذج المضافة في المتصفح، فلا داعي لإعادة إضافتها عند تحديث الصفحة.

## خارطة الطريق والمقترحات المستقبلية (Roadmap)

فيما يلي قائمة بالميزات المخطط لها لتحسين التطبيق:

1.  **دعم تخزين السياق (Context Caching Calculator)**:
    *   إضافة خيار لحساب تكلفة الـ Cached Input (التي تكون أرخص عادةً بنسبة 75%) وتكلفة التخزين بالساعة.

2.  **محاكي الميزانية (Reverse Budget Calculator)**:
    *   إدخال الميزانية الشهرية (مثلاً 50$) ليقوم التطبيق بحساب الحد الأقصى للطلبات المسموح بها يومياً.

3.  **مقارنة "الحلبة" (Arena Mode)**:
    *   تشغيل نموذجين في نفس الوقت (Side-by-side) لمقارنة التكلفة والجودة والاستجابة الزمنية.

4.  **تصدير التقارير (Export Reports)**:
    *   تصدير جداول التكلفة بصيغة CSV أو PDF لمشاركتها مع فرق العمل.

5.  **دعم الوسائط المتعددة المتقدم**:
    *   حساب تكلفة الفيديو بالثانية وتكلفة الصوت بشكل أكثر تفصيلاً في الواجهة الحية.

## كيفية الاستخدام

### 1. الإعداد الأولي
تأكد من وجود مفتاح API صالح في متغيرات البيئة (`process.env.API_KEY`). التطبيق يعتمد على حزمة `@google/genai`.

### 2. إضافة نموذج جديد
1.  اضغط على زر "إضافة نموذج" (+ Add Model) في أعلى الصفحة.
2.  استخدم البحث الذكي أو أدخل البيانات يدوياً.
3.  سيتم حفظ النموذج تلقائياً لاستخدامه في المرات القادمة.

### 3. حساب التكلفة
*   استخدم "مساعد التحويل" لتحويل عدد كلمات مستندك إلى توكنز تقريبية.
*   قارن التكلفة بين النماذج المختلفة في الرسم البياني.
